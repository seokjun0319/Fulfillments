# fulfillment-info (prototype)

이지메디컴 풀필먼트 사업부 정보공유 페이지 **UI 뼈대(모양만)** 입니다.

## 실행 방법

- `index.html`을 브라우저로 열면 됩니다.
  - 로컬 파일로 열어도 동작합니다(정적 JS).
  - (권장) 로컬 서버(Live Server 등)로 열면 미리보기/새로고침이 편합니다.

## 구성

- `index.html`: 레이아웃(좌측 메뉴 + 우측 콘텐츠)
- `styles.css`: 기본 스타일(다크 톤)
- `app.js`: 해시 기반 탭 전환 + 더미 콘텐츠 렌더링
- `assets/logo.svg`: 로고(임시). 실제 회사 로고 SVG/PNG로 교체하면 됩니다.
